# OpenZeppelin GSNV1 Example

## Quick Start

Insert command into the terminal:

```bash
$ git clone https://github.com/ilyakmet/openzeppelin-gsn-example.git && cd ./openzeppelin-gsn-example && yarn && cd ./client && yarn && yarn start
```
